# Chemical Dosage Calculators

## Standard Formulas
This section tracks the approved formulas for daily chemical feed rates.

| Chemical | Target Dose (mg/L) | Feed Rate Formula |
| :--- | :--- | :--- |
| **Chlorine** | 2.5 | `(Flow * 8.34 * Dose) / Strength` |
| **Fluoride** | 0.7 | *Pending validation* |
